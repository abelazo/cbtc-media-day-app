# Tutorials

**Learning-oriented.** Lessons that take a newcomer by the hand through a series of steps to complete a project, so they gain confidence and basic competence.

- Must work, every time, for every reader.
- Get the reader to a result quickly; avoid explaining alternatives or why.
- No choices for the reader to make — one path.

Example candidates: "Set up your local dev environment and run the app end-to-end", "Deploy your first change to `dev`".

See https://diataxis.fr/tutorials/
